R4P by bwlewis

Tools for foolin' with R and Processing together.

R4P provides basic tools for foolin' with R and Processing together. One may call R code from Processing code and vice-versa. R4P includes a basic Processing-based graphics output device for R.

Dependencies

Processing: http://processing.org/ R: http://www.r-project.org/ rJava: http://www.rforge.net/rJava/

Install

1. Install Processing. 2. Install R. 3. Install the rJava R package (for example, from within R, use 'install.packages("rJava").' 4. Download R4P 5. Set the PROCESSING environment variable to your Processing installation path. 6. Install the R4P package from R, for example from the command line with: 'R CMD INSTALL R4P.' 7. Start R4P with it's convenience script as described by the installation message. 8. Open the example sketches (boids and SimpleR4P) and start foolin'.
